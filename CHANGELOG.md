# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.1.0] - 2026-01-06

### Added
- Added blocklist
- Added glint to enabled trolls
- Added command to add and remove players to block list
- Added option in config to alow/disallow to troll OP players
- Added command to enable or disable the above option in game
- Added RandomCraft troll

### Fixed

- Fixed a random error on command throwing up on console
- Fixed herobrine skin not showing on head item on troll menu

### Changed

- Changed structure hierarchy in project folder
- Renamed some stuff for fun

## [3.0.0] - 2026-01-02

### Added

- Added right click to disable troll
- Added switch player on all gui
- Added messages on trolling / untrolling
- support for 1.21.11

### Fixed

- Fixed some trolls not being functional at all on porting to 1.21.11
- Added a temporary workaround for world loading

### Changed

- Changed ui title to show the player you are trolling
- Made herobrine troll show herobrine's head on online mode servers
- added lore to all items about enabling and disabling a troll

### Removed

- Removed chinese translation from the original plugin
- removed xseries files locally from project to implement it as a dependency in gradle
